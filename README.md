# Microservices with CQRS and Event Sourcing

Ce projet est une architecture microservices basée sur **CQRS** et **Event Sourcing** utilisant Spring Boot et Axon Framework.

**Auteur** : Aimane Maane
**Dépôt d'origine personnalisé** : https://github.com/maane2025/microservices-with-cqrs-event-sourcing

## Technologies
- Java 17
- Spring Boot
- Axon Framework
- H2 Database (In-Memory)
- Maven

## Services
- **Account-Service** : Gestion des comptes (Port 8084)
- **Analytics-Service** : Analyse des données (Port 8085)
